📋 Schritt-für-Schritt Anleitung: WordPress
  1. Bei WordPress einloggen
  2. Gehen Sie zu Benutzer → Ihr Profil
  3. Notieren Sie Ihren Benutzernamen ganz oben auf der Seite – tragen Sie diesen im Feld „Ihr WordPress-Benutzername“ ein
  4. Scrollen Sie runter zu Anwendungspasswörter
  5. Geben Sie einen beliebigen Namen (z.B. Statlift) ein und klicken Sie Hinzufügen
  6. Kopieren Sie das generierte Passwort – es wird nur einmal angezeigt!
  7. Tragen Sie es oben als App-Passwort ein
⚠️ Benötigt: Scopes cms.blogs.read + cms.blogs.write.
📋 Schritt-für-Schritt Anleitung: HubSpot
  1. HubSpot Dashboard öffnen
  2. Gehen Sie zu Einstellungen → Integrationen → Private Apps
  3. Klicken Sie auf Private App erstellen
  4. Name: Statlift, Scopes: cms.blogs.read + cms.blogs.write
  5. App erstellen → Access Token kopieren
  6. Blog-ID: Marketing → Website → Blog → ID aus der URL
⚠️ Token muss Scope cms:read + cms:write haben.
📋 Schritt-für-Schritt Anleitung: Webflow
  1. Webflow Dashboard öffnen
  2. Ihre Website auswählen → Site Settings
  3. Navigieren Sie zu Apps & Integrations → API Access
  4. Klicken Sie auf Generate API Token
  5. Scopes: cms:read + cms:write wählen
  6. Token sofort kopieren – wird nicht erneut angezeigt!
  7. Collection ID: Im Designer → CMS → Blog-Collection → ID aus URL
⚠️ Benötigt Extension EXT:headless oder EXT:rest.
📋 Schritt-für-Schritt Anleitung: TYPO3
  1. TYPO3 Backend öffnen
  2. Extension EXT:headless installieren (Extension Manager)
  3. Im Backend: System → API-Tokens
  4. Neuen Token erstellen mit Schreibberechtigung
  5. URL und Token oben eintragen
⚠️ Benötigt API-Bundle (z.B. oneup/contao-api-bundle).
📋 Schritt-für-Schritt Anleitung: Contao
  1. Contao Manager öffnen
  2. Extension oneup/contao-api-bundle installieren
  3. Im Contao Backend: System → API-Apps
  4. Neue App erstellen mit Schreibrechten für News-Archive
  5. API-Schlüssel kopieren und oben eintragen
⚠️ REST-Modul + Basic Auth-Modul müssen aktiviert sein.
📋 Schritt-für-Schritt Anleitung: Drupal
  1. Drupal Admin öffnen
  2. Verwaltung → Module: RESTful Web Services + Basic Auth aktivieren
  3. Konfiguration → Web Services → REST → Content-Endpunkt aktivieren
  4. Benutzer mit Berechtigung Inhalte erstellen verwenden
  5. URL, Benutzername und Passwort oben eintragen
Wir senden dir so schnell wie möglich eine Anfrage über Shopify. Bitte nimm diese an. Sobald das geschehen ist, richten wir alles für dich ein.
📋 Schritt-für-Schritt Anleitung: Shopify
  1. Shopify Admin öffnen
  2. Gehe zu Einstellungen → Nutzer und Berechtigungen → Sicherheit
  3. Kopiere dort deinen 4-stelligen Collaborator-Code (Mitarbeiter-Code)
  4. Shop-URL und 4-stelligen Code hier eintragen
⚠️ Integration unter Einstellungen > System > Integrationen erstellen.
📋 Schritt-für-Schritt Anleitung: Shopware
  1. Shopware Admin öffnen
  2. Einstellungen → System → Integrationen
  3. Integration hinzufügen → Name: Statlift
  4. CMS-Berechtigungen aktivieren
  5. Access Key ID + Secret Access Key kopieren
ℹ️ Folgen Sie der Anleitung unten, um Ihren Wix API-Schlüssel zu erhalten und Ihre Website-URL einzufügen.
Bitte geben Sie Ihre Wix-URL oder Site-ID ein.
📋 Schritt-für-Schritt Anleitung: Wix
  1. Öffnen Sie Ihre Wix-Website im Editor und klicken Sie im linken Menü auf Apps
  2. Klicken Sie auf App hinzufügen, suchen Sie nach Wix Blog und fügen Sie die App hinzu (falls noch nicht vorhanden)
  3. Gehen Sie zu den Einstellungen Ihrer Website → Headless-Einstellungen → API-Schlüssel verwalten
  4. Klicken Sie auf API-Schlüssel generieren, setzen Sie unter Website-Berechtigungen einen Haken bei Wix Blog und klicken Sie auf Schlüssel generieren
  5. Kopieren Sie den generierten API-Schlüssel in das Feld oben
  6. Gehen Sie nun zu Ihrem Wix Dashboard (manage.wix.com), wählen Sie Ihre Website aus und kopieren Sie die gesamte URL aus der Adresszeile Ihres Browsers in das Feld „Website-URL oder Site-ID" – wir extrahieren die ID automatisch
⚠️ Unter Einstellungen > Entwicklertools > API-Schlüssel.
📋 Schritt-für-Schritt Anleitung: Squarespace
  1. Squarespace Dashboard öffnen
  2. Einstellungen → Entwicklertools → API-Schlüssel
  3. API-Schlüssel generieren
  4. Berechtigungen: Website → Inhalte lesen/schreiben
  5. Schlüssel kopieren und oben eintragen
