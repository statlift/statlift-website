📋 Step-by-Step Guide: WordPress
  1. Log into WordPress
  2. Go to Users → Profile
  3. Scroll down to Application Passwords
  4. Enter Statlift and click Add New Application Password
  5. Copy the generated password – it will only be shown once!
  6. Enter it above as your App Password
⚠️ Requires: Scopes cms.blogs.read + cms.blogs.write.
📋 Step-by-Step Guide: HubSpot
  1. Open your HubSpot Dashboard
  2. Go to Settings → Integrations → Private Apps
  3. Click on Create a private app
  4. Name: Statlift, Scopes: cms.blogs.read + cms.blogs.write
  5. Create app → Copy Access Token
  6. Blog ID: Marketing → Website → Blog → ID from the URL
⚠️ Token must have Scope cms:read + cms:write.
📋 Step-by-Step Guide: Webflow
  1. Open your Webflow Dashboard
  2. Select your site → Site Settings
  3. Navigate to Apps & Integrations → API Access
  4. Click Generate API Token
  5. Select Scopes: cms:read + cms:write
  6. Copy token immediately – it will not be shown again!
  7. Collection ID: In Designer → CMS → Blog Collection → ID from URL
⚠️ Requires Extension EXT:headless or EXT:rest.
📋 Step-by-Step Guide: TYPO3
  1. Open TYPO3 Backend
  2. Install Extension EXT:headless (Extension Manager)
  3. In Backend: System → API Tokens
  4. Create new token with write access
  5. Enter URL and token above
⚠️ Requires API Bundle (e.g., oneup/contao-api-bundle).
📋 Step-by-Step Guide: Contao
  1. Open Contao Manager
  2. Install Extension oneup/contao-api-bundle
  3. In Contao Backend: System → API Apps
  4. Create new app with write permissions for News Archives
  5. Copy API key and enter it above
⚠️ REST module + Basic Auth module must be enabled.
📋 Step-by-Step Guide: Drupal
  1. Open Drupal Admin
  2. Manage → Extend: Enable RESTful Web Services + Basic Auth
  3. Configuration → Web Services → REST → Enable Content Endpoint
  4. Use a user with Create Content permissions
  5. Enter URL, Username, and Password above
⚠️ Token requires permission: read_content + write_content.
📋 Step-by-Step Guide: Shopify
  1. Open Shopify Admin
  2. Apps → App and sales channel settings → Develop apps
  3. Create an app → API credentials
  4. Admin API scopes: read_content + write_content
  5. Install app → Copy Admin API access token
  6. Blog ID: Online Store → Blog Posts → Manage Blogs → Click Blog → ID from URL
⚠️ Create integration under Settings > System > Integrations.
📋 Step-by-Step Guide: Shopware
  1. Open Shopware Admin
  2. Settings → System → Integrations
  3. Add integration → Name: Statlift
  4. Enable CMS permissions
  5. Copy Access Key ID + Secret Access Key
⚠️ API Key Manager in Wix Account Dashboard.
📋 Step-by-Step Guide: Wix
  1. Open Wix Account Dashboard (not Editor)
  2. Profile picture → Account Settings → API Keys
  3. Generate API Key → Name: Statlift
  4. Select Blog permissions → Copy key
  5. Site ID: In your dashboard URL after /dashboard/
⚠️ Under Settings > Developer Tools > API Keys.
📋 Step-by-Step Guide: Squarespace
  1. Open Squarespace Dashboard
  2. Settings → Developer Tools → API Keys
  3. Generate API Key
  4. Permissions: Website → Read/Write Content
  5. Copy key and enter it above
